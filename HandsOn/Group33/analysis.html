Analysis:

 

1. Analyse Data Set

El CSV presenta todos los datos de lugares de estacionamiento y carga para coches electricos en la ciudad de Madrid. Representado:

- Distrito donde se encuentra (string)

- Direccion (string)

- Emplazamiento (string)-

- Numero de cargadores (int)

- Operador que instalo dicho punto de carga (string)

- Coordenadas eje X (string)

- Coordenadas eje y (string)

- Longitud (string)

- Latitud (string)

2. Analyse Licensing of the Data Source

El autor es la propia Comunidad de Madrid, dentro de la parte dedicada a proporcionar datos abiertos de sus diversos servicios.

La licencia viene descrita dentro de este link: Licencia
